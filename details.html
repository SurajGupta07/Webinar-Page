  • Abstract:
  • Creating Visibility is one of the key aspects of any Cyber Defense program. Once the visibility is created, it is important to create the detection logic on top of the visibility that is created.
  • MITRE’s ATT&CK framework has been gaining a lot of popularity in the past few years and has become the de-facto standard for building cyber-threat-detection capabilities. ATT&CK is a global repository for various techniques seen in-the-wild by the adversaries/threat actors at each stage of the cyber-attack kill chain. With MITRE’s ATT&CK, we get more than 300 cyber-attack indicators that can be leveraged to build cyber-attack detection capabilities.
  • Before MITRE’s ATT&CK framework, “what to look for” was always been a challenge for the organizations as there was no single comprehensive source/framework that can be used to provide the direction to build the detection building programs.
  • In this webinar, we will look at an overview of the ATT&CK framework and discuss forward-path on how to put this framework into action.
